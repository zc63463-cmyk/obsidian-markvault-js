## 一、核心命题（v3 最终版）

### 三版命题演化

| 版本 | 命题 | 产品形态 |
|------|------|---------|
| v1 | 为深度学习者，在 PDF 原文标注与 Markdown 笔记标注之间建立语义关系图谱的 Obsidian 插件 | 插件 |
| v2 | 为小学→大学的学生，建立个人认知网络，通过深度标注×语义关系×AI Agent 分析实现认知闭环 | 插件 + AI Agent |
| **v3** | **为终身学习者，建立个人认知操作系统——通过"深度标注 × 关系网络 × 主动梳理 × AI 洞察"四维协同，让每个人在 AI 时代拥有自己的认知工具。** | **插件 + MindFlow + AI Agent** |

### v3 关键增量：MindFlow 补全"主动梳理"维度

v2 的三维体系（标注 × 关系 × AI）缺少一个关键环节——**人主动加工认知的能力**。MindFlow 不是"又一个视图"，而是补全了第四维：

```
深度标注 (MarkVault) → 关系发现 (RelationGraph) → 主动梳理 (MindFlow) → AI 洞察 (Agent)
     ↑                                                                   |
     └────────────── 回到原文补充 ←──── 发现遗漏 ←──────────────────────┘
```

**RelationGraph = 被动的全景呈现（看见有什么）**
**MindFlow = 主动的结构化加工（讲清为什么）**

两者互补：看见 → 理解 → 表达 → 反馈，构成完整的认知闭环。

## 六、四维体系完整论证

### 6.1 深度标注（MarkVault 核心）— 数据产生层

v1.2 校准后的设计更简洁：
- **PDF 标注 text 默认留空**——用户截图→豆包→MD→粘贴，零额外字段，零额外渲染成本
- **2 种标注形式**——荧光笔 + 方框区域，覆盖 80%+ 场景，画笔延后
- **70% 代码直接复用**——关系引擎、索引层、W3C 兼容层、分片持久化零改动

### 6.2 关系网络（RelationGraph）— 数据连接层

- **25 种语义关系**，6 维度分层着色（分类/论证/阐释/指涉/因果/结构）
- **Canvas 力导向图** + computeCurvature 弧线 + 多维过滤
- **AI 可分析/整理/扩展**的原始数据网络
- 定位：**被动的全景呈现**——看见全局关系，发现隐含关联

### 6.3 主动梳理（MindFlow）— 认知加工层

- **RelationGraph 的补充**，不是替代
- **温故知新**——在 RG 中看见关联 → 在 MF 中主动提取重组 → 发现遗漏 → 回来补充
- **费曼学习法**——导图 = 讲给别人的框架，清晰的思维框架 + 标注节点跳转 + 详情
- 定位：**主动的结构化加工**——讲清一个方面的逻辑

### 6.4 AI 洞察（Agent）— 智能扩展层

- **纠偏**：needsCorrection + confidence 低 → 检测理解偏差
- **补充**：孤立节点 → 发现隐含关联
- **拓展**：基于认知指纹推荐新概念
- **学习路径**：基于 mastery 时序 + relation 路径优化
- **错题归因**：习题标注 + motivation=editing + 关联知识点 → 根因分析
- **订阅制**：¥15-30/月 基础，¥50-100/月 Pro
