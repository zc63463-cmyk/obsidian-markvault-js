### 关键经验

1. **re-export 陷阱**: `export { X } from './module'` 不创建本地绑定，内部调用需单独 `import`
2. **提取 private→export 函数**: 必须逐行对比原始实现，重写会丢失逻辑（parseMarkTags 丢了 4 处）
3. **不同 kind 写入 MD 的字段不同**: native 只写 color/type，region 不写 alias，store-only 警告需按 kind 判断
