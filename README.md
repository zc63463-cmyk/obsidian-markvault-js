# obsidian-markvault

Powerful annotation plugin for Obsidian: highlights, bold, underline with notes, tags, and a sidebar panel for managing annotations across your vault.

## Features

- 🎨 **Multi-style annotations**: Highlight, Bold, Underline
- 📝 **Annotation notes**: Add notes/comments to any annotation
- 🏷️ **Tags**: Organize annotations with tags
- 📊 **Sidebar panel**: Browse, search, and filter annotations
- 🔗 **Click-to-jump**: Click annotation in sidebar to jump to source text
- 💾 **Dual storage**: IndexedDB for fast queries + Markdown inline for portability
- 🔄 **Offset recovery**: 4-layer algorithm keeps annotations accurate after edits

## Installation

1. Download the latest release
2. Extract into your vault's `.obsidian/plugins/obsidian-markvault` directory
3. Enable the plugin in Obsidian settings

## Development
